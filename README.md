# SentiScope

SentiScope is an AI-powered feedback analyzer designed to help teams understand sentiment, themes, and customer intent from reviews, surveys, and support feedback.

## Features
- Collect and organize feedback text
- Analyze sentiment and emotional tone
- Highlight recurring themes and issues
- Prepare insights for product and support teams

## Project Status
This repository has been initialized as a starter project for the SentiScope idea.

## Getting Started
1. Clone the repository
2. Install dependencies if needed
3. Run the starter script

```bash
python app.py
```
